2.2.5. Move an Existing Page

The user needs at least 2 sections in an existing class, as well as a page that needs to be moved from one section to another.


The user right clicks on the tutorial or quiz they want to move. The right click menu appears again.


The user can then select move. The program will then open the move window


The user selects what section they want to move the page to via the dropdown menu, then press OK. The program will then move the class to the desired location. In the image, the user selected 'Variables.' The moved page can be seen below:
