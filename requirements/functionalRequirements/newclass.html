2.2.1. Adding a new class

To view the manager screen, the user must navigate from the home page after logging in. It is only available to professors and administrators. Once logged in, they are able to view the manager.


To create a new class, the user should click on the 'Create New Class' button. The program displays the following screen:


The user then types the name of the class in the text field currently filled as 'Untitled'. They can also select who can view tutorials in the class. The 3 options are anyone who has a link to the page, only students in the class, or only the professor. When finished, the user should click the 'OK' button.


The user should see the new class with the title they entered into the field.
