2.2.3. Adding a new tutorial or quiz

The user should be on the manager screen with a class already created, as well as at least 1 section in the class


Tutorials and quizzes go into larger sections. To create a new quiz, or tutorial, the user right clicks on the class area in the second panel. The program displays the right click menu:


To create a new page, the user clicks on the 'Create Page' option. The program displays the following screen:


The user can now name the page. They can select whether the page should be a quiz or a tutorial. When finished, the user should click the 'OK' button. The user will be taken to the editor for quizzes LINKME or tutorials LINKME. The manager screen shows the new page after the user returns from editing.
